# 🔗 Encurtador de URL

![Encurtador de URL Cover](https://i.imgur.com/NGLyRoM.png)

## 🎯 Objetivo
Criação de uma aplicação WEB para que a pessoa consiga encurtar links que sejam grandes demais.

## ⚙️ Funcionalidades
- Encurtar links
- Armazenamento de links já encurtados
